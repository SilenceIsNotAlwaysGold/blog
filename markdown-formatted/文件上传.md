# 上传文件：
要将本地文件上传到云服务器，你可以使用以下方法之一：

1. 使用 scp 命令：
    - 从本地复制文件到云服务器：
    - 从云服务器复制文件到本地：
2. 例如，要将本地的 example.txt 文件上传到云服务器的 /home/username 目录下，可以使用以下命令：

```ruby
scp /path/to/local/file username@server_ip:/path/to/destination
```

```ruby
scp username@server_ip:/path/to/remote/file /path/to/destination
```

```ruby
scp /path/to/example.txt username@server_ip:/home/username
```

3. 使用 rsync 命令：
    - 从本地同步文件到云服务器：
    - 从云服务器同步文件到本地：
4. 例如，要将本地的 example.txt 文件同步到云服务器的 /home/username 目录下，可以使用以下命令：

```ruby
rsync -avz /path/to/local/file username@server_ip:/path/to/destination
```

```ruby
rsync -avz username@server_ip:/path/to/remote/file /path/to/destination
```

```ruby
rsync -avz /path/to/example.txt username@server_ip:/home/username
```

请将上述命令中的 username 替换为你的云服务器的用户名，server_ip 替换为云服务器的 IP 地址，/path/to/local/file 替换为本地文件的路径，/path/to/destination 替换为上传到云服务器的目标路径。
