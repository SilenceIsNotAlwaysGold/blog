1、打开cmd，安装⼯具包  
pip install virtualenv  
pip install virtualenvwrapper-win

如果是linux或mac系统，执⾏安装pip install virtualenvwrapper


2、添加环境变量，win系统-我的电脑-右键属性-⾼级设置-系统变量-新建环境变量  
变量名：WORKON_HOME  
变量值：⾃⼰在系统磁盘⽬录新建⽂件夹，⽤来保存虚拟环境的⽬录  
#######################################################  
把cmd关掉，重新打开cmd  
#######################################################  
3、创建虚拟环境，必须联⽹  
mkvirtualenv -p python3 django_py3  
4、如果执⾏成功，会⾃动进⼊虚拟环境中，django_py3


查看当前环境中，安装了哪些⼯具包

pip list


# 进⼊虚拟环境


workon 虚拟环境名称  
workon django_py3


# 退出虚拟环境


deactivate


# 把当前环境的模块，全部打包整理成依赖⽂件


pip freeze > requirements.txt


# 安装依赖⽂件


pip install -r requirements.txt


# 安装过程中，如果安装较慢可以使⽤第三⽅源


pip install flask -i [https://mirrors.aliyun.com/pypi/simple/](https://mirrors.aliyun.com/pypi/simple/)


# 安装django并指定版本


pip install django # 默认按照最新版  
pip install django==3.2.3 # 指定版本安装


更新pip的版本  
python -m pip install --upgrade pip
