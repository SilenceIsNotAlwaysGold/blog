1.  **均方误差 (MSE)** 
    - **应用场景**：用于回归问题，如房价预测、温度预测等。
    - **判断标准**：MSE 越低，表示模型预测越准确。
    - **示例代码**:

```python
from sklearn.metrics import mean_squared_error

y_true = [3.0, 4.0, 5.0, 6.0, 7.0]
y_pred = [2.8, 3.9, 5.2, 6.5, 7.2]

mse = mean_squared_error(y_true, y_pred)
print(f'MSE: {mse:.2f}')
```

 

2.  **平均绝对误差 (MAE)** 
    - **应用场景**：同样用于回归问题，尤其当数据中存在异常值时。
    - **判断标准**：MAE 越低，表示模型预测越准确。
    - **示例代码**:

```python
from sklearn.metrics import mean_absolute_error

y_true = [3.0, 4.0, 5.0, 6.0, 7.0]
y_pred = [2.8, 3.9, 5.2, 6.5, 7.2]

mae = mean_absolute_error(y_true, y_pred)
print(f'MAE: {mae:.2f}')
```

 

3.  **R²系数** 
    - **应用场景**：用于回归问题，衡量模型解释数据变异的能力。
    - **判断标准**：R²越接近1，表示模型解释了大部分数据的变异，预测性能越好。
    - **示例代码**:

```python
from sklearn.metrics import r2_score

y_true = [3.0, 4.0, 5.0, 6.0, 7.0]
y_pred = [2.8, 3.9, 5.2, 6.5, 7.2]

r2 = r2_score(y_true, y_pred)
print(f'R²: {r2:.2f}')
```

 

4.  **对数损失 (Log Loss)** 
    -  **应用场景**：用于分类问题，特别是在需要评估预测概率准确性的情况下。 
    -  **判断标准**：Log Loss 越低，表示模型预测越准确。 
    -  **示例代码**: 

```python
from sklearn.metrics import log_loss

y_true = [1, 0, 1, 1, 0]
y_pred = [0.9, 0.1, 0.8, 0.65, 0.3]

logloss = log_loss(y_true, y_pred)
print(f'Log Loss: {logloss:.2f}')
```

 

5.  **精确度 (Precision) 和 召回率 (Recall)** 
    - **应用场景**：用于分类问题，尤其适用于需要权衡误报和漏报的情况，例如医疗诊断或欺诈检测。
    - **判断标准**：高精确度表示模型在预测为正例的样本中有更高比例的真正例；高召回率表示模型能识别更多实际正例。
    - **示例代码**:

```python
from sklearn.metrics import precision_score, recall_score

y_true = [1, 0, 1, 1, 0, 1]
y_pred = [1, 1, 0, 1, 0, 0]

precision = precision_score(y_true, y_pred)
recall = recall_score(y_true, y_pred)

print(f'Precision: {precision:.2f}')
print(f'Recall: {recall:.2f}')
```

 

6.  **F1分数** 
    - **应用场景**：用于分类问题，特别是在精确度和召回率之间需要平衡的情况，如不平衡数据集。
    - **判断标准**：F1分数越高，表示模型在精确度和召回率之间取得了更好的平衡。
    - **示例代码**:

```python
from sklearn.metrics import f1_score

y_true = [1, 0, 1, 1, 0, 1]
y_pred = [1, 1, 0, 1, 0, 0]

f1 = f1_score(y_true, y_pred)
print(f'F1 Score: {f1:.2f}')
```

 

7.  **ROC曲线和AUC** 
    - **应用场景**：用于分类问题，尤其适用于评估模型在不同阈值下的性能，例如在处理不平衡类别数据时。
    - **判断标准**：AUC值越接近1，表示模型的分类性能越好。
    - **示例代码**:

```python
from sklearn.metrics import roc_auc_score


y_true = [1, 0, 1, 1, 0]
y_pred = [0.9, 0.1, 0.8, 0.65, 0.3]

auc = roc_auc_score(y_true, y_pred)
print(f'AUC: {auc:.2f}')
```

 

8.  **混淆矩阵** 
    - **应用场景**：用于分类问题，特别是当需要详细分析模型的性能，包括对不同类别的预测准确性。
    - **判断标准**：通过比较真正例、假正例、真负例和假负例的数量，可以详细了解模型性能。
    - **示例代码**:

```python
from sklearn.metrics import confusion_matrix

y_true = [1, 0, 1, 1, 0, 1]
y_pred = [1, 1, 0, 1, 0, 0]

cm = confusion_matrix(y_true, y_pred)
print('Confusion Matrix:')
print(cm)
```

 


每种评估方法都有其特定的适用场景和评估标准。在实际应用中，通常会根据具体问题的需求和性质选择适当的评估方法。例如，回归问题通常使用MSE、MAE或R²系数，而分类问题则依赖于诸如精确度、召回率、F1分数和AUC等指标。混淆矩阵则提供了一种直观的方式来查看分类模型在不同类别上的表现。
