1. **初始化仓库**：


```bash
git init
```


这个命令用于在当前目录下初始化一个新的 Git 仓库。


2. **克隆远程仓库**：


```bash
git clone 
```


使用此命令可以将远程仓库克隆到本地。


3. **添加文件到暂存区**：


```bash
git add 
```


将指定文件添加到 Git 的暂存区中。


4. **提交更改**：


```bash
git commit -m "Commit message"
```


提交暂存区的文件并添加提交信息。


5. **推送更改到远程仓库**：


```bash
git push origin 
```


将本地分支的更改推送到远程仓库。


6. **拉取远程仓库的最新更改**：


```bash
git pull origin 
```


从远程仓库拉取最新的更改并合并到本地分支。


7. **查看文件状态**：


```bash
git status
```


查看工作目录的文件状态，包括已修改的文件、暂存的文件等。


8. **查看提交历史**：


```bash
git log
```


查看提交历史，显示所有提交记录。


9. **创建分支**：


```bash
git branch 
```


创建一个新的分支。


10. **切换分支**：


```bash
git checkout 
```


切换到指定的分支。


11. **合并分支**：


```bash
git merge 
```


将指定分支合并到当前分支。


12. **查看远程仓库列表**：


```bash
git remote -v
```


查看远程仓库的 URL 列表。
