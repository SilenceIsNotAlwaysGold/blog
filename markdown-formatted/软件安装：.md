系统是基于Red Hat或CentOS，可以使用yum包管理器来安装pip和Flask。使用以下命令安装pip：

```plain
sudo yum install python3-pip
```

然后使用pip来安装Flask：

```plain
pip3 install flask
```


系统是基于Arch Linux，可以使用pacman包管理器来安装pip和Flask。使用以下命令安装pip：

```plain
sudo pacman -S python-pip
```

然后使用pip来安装Flask：

```plain
pip3 install flask
```
