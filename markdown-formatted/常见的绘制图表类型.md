1.  **条形图（Bar Chart）**: 
    - **应用场景**：比较不同类别的数量、频率等。
    - **示例代码**：

```python
import matplotlib.pyplot as plt
import seaborn as sns

categories = ['Category A', 'Category B', 'Category C']
values = [23, 45, 56]

sns.barplot(x=categories, y=values)
plt.xlabel('Categories')
plt.ylabel('Values')
plt.title('Bar Chart Example')
plt.show()
```

 

2.  **折线图（Line Chart）**: 
    - **应用场景**：展示数据随时间或有序类别的趋势。
    - **示例代码**：

```python
import matplotlib.pyplot as plt

time = ['January', 'February', 'March', 'April']
values = [23, 45, 56, 78]

plt.plot(time, values)
plt.xlabel('Month')
plt.ylabel('Value')
plt.title('Line Chart Example')
plt.show()
```

 

3.  **散点图（Scatter Plot）**: 
    - **应用场景**：展示两个连续变量之间的关系。
    - **示例代码**：

```python
import matplotlib.pyplot as plt

x_values = [5, 7, 8, 7, 2, 17, 2, 9, 4, 11, 12, 9, 6]
y_values = [99, 86, 87, 88, 100, 86, 103, 87, 94, 78, 77, 85, 86]

plt.scatter(x_values, y_values)
plt.xlabel('X Value')
plt.ylabel('Y Value')
plt.title('Scatter Plot Example')
plt.show()
```

 

4.  **饼图（Pie Chart）**: 
    - **应用场景**：显示部分与整体之间的比例。
    - **示例代码**：

```python
import matplotlib.pyplot as plt

sizes = [25, 35, 15, 25]
labels = ['Category A', 'Category B', 'Category C', 'Category D']

plt.pie(sizes, labels=labels, autopct='%1.1f%%')
plt.axis('equal')  # Equal aspect ratio ensures the pie chart is circular.
plt.title('Pie Chart Example')
plt.show()
```

 

5.  **箱形图（Box Plot）**: 
    - **应用场景**：显示数据分布的统计概括，如中位数、四分位数和异常值。
    - **示例代码**：

```python
import seaborn as sns

data = [23, 45, 56, 78, 33, 44, 55, 76, 29, 47]

sns.boxplot(data=data)
plt.title('Box Plot Example')
plt.show()
```

 

6.  **直方图（Histogram）**: 
    - **应用场景**：展示连续变量的分布情况。
    - **示例代码**：

```python
import matplotlib.pyplot as plt

data = [23, 45, 56, 78, 33, 44, 55, 76, 29, 47, 59, 60, 70, 75, 90]

plt.hist(data, bins=5)
plt.xlabel('Value')
plt.ylabel('Frequency')
plt.title('Histogram Example')
plt.show()
```

 

7.  **热力图（Heatmap）**: 
    - **应用场景**：显示两个类别变量之间关系的强度或数值变量之间的相关性。


+ **示例代码**：

```python
import seaborn as sns; sns.set_theme()
import numpy as np

# 创建一个10x10的随机数据矩阵
data = np.random.rand(10, 10)

sns.heatmap(data, annot=True)
plt.title('Heatmap Example')
plt.show()
```

 


:::success
这些图表类型在数据可视化中被广泛使用，它们各自有不同的用途和适用场景。例如，条形图和饼图适合用于显示类别数据，折线图和直方图适合展示连续数据，而散点图和热力图则适合用于探索变量之间的关系。选择哪种图表类型取决于您的数据特性和您想要传达的信息。使用Python的matplotlib和seaborn等库可以方便地创建这些图表。

:::
