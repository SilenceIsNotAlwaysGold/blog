# 用户创建：
在 Linux 系统中，要添加用户，可以使用以下命令：

1. 使用 useradd 命令：

```plain
sudo useradd username
```

这将创建一个新的用户，用户名为 username。默认情况下，useradd 命令会在 /home 目录下创建与用户名相同的用户目录。

2. 使用 adduser 命令：

```plain
sudo adduser username
```

adduser 命令是 useradd 命令的一个更友好的前端界面，它会提示你输入一些用户信息，例如密码、全名等。

3. 设置用户密码：

```plain
sudo passwd username
```

这将为用户设置密码。你需要输入两次密码以确认。

4. 为用户添加用户组：

```plain
sudo usermod -aG groupname username
```

这将将用户添加到指定的用户组中。将 groupname 替换为用户组的名称，将 username 替换为要添加到用户组的用户名。

5. 切换到新用户：

```plain
sudo su - username
```

这将切换到新创建的用户。你可以使用这个命令来测试新用户是否正常工作。

请注意，在使用以上命令时，需要使用具有管理员权限的用户执行。你可以使用 sudo 命令来提升权限。另外，还可以使用其他选项和参数来进一步自定义用户的创建过程，可以通过 man useradd 或 man adduser 命令查看相关文档来获取更多信息。
