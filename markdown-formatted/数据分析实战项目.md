# 数据分析实战项目：电商销售数据分析

在学习了 Pandas、NumPy 和 Matplotlib 的基础知识后，我们通过一个虚构的电商销售数据集来实战演练。我们的目标是回答以下商业问题：
1. 哪个月份的销售额最高？
2. 哪个城市的订单量最大？
3. 什么时间段是用户下单的高峰期？
4. 哪种产品最畅销？

## 1. 准备工作

首先，我们模拟生成一份数据（在实际项目中，你会直接读取 `sales_data.csv`）。

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# 模拟数据生成
np.random.seed(42)
dates = pd.date_range(start='2025-01-01', periods=1000, freq='H')
products = ['Laptop', 'Mouse', 'Monitor', 'Keyboard', 'Headphones']
cities = ['New York', 'Los Angeles', 'Chicago', 'San Francisco']

data = {
    'Order ID': range(1, 1001),
    'Date': np.random.choice(dates, 1000),
    'Product': np.random.choice(products, 1000),
    'Quantity': np.random.randint(1, 5, 1000),
    'Price': np.random.choice([1000, 20, 200, 50, 80], 1000), # 简化价格逻辑
    'City': np.random.choice(cities, 1000)
}
# 修正价格：让价格与产品对应 (简单映射)
price_map = {'Laptop': 1000, 'Mouse': 20, 'Monitor': 200, 'Keyboard': 50, 'Headphones': 80}
df = pd.DataFrame(data)
df['Price'] = df['Product'].map(price_map)

# 添加一些脏数据用于清洗演示
df.loc[0:5, 'Quantity'] = np.nan # 缺失值

# 计算总价
df['Total Sales'] = df['Quantity'] * df['Price']

print(df.head())
```

## 2. 数据清洗

```python
# 1. 检查缺失值
print("缺失值统计:\n", df.isnull().sum())

# 2. 处理缺失值 (删除)
df = df.dropna()

# 3. 确保数据类型正确
# 这里的 Date 已经是 datetime 类型，如果是读取 CSV 可能是字符串
# df['Date'] = pd.to_datetime(df['Date'])

print("清洗后数据形状:", df.shape)
```

## 3. 数据分析与回答问题

### Q1: 哪个月份的销售额最高？

我们需要从 `Date` 列提取月份。

```python
df['Month'] = df['Date'].dt.month

monthly_sales = df.groupby('Month')['Total Sales'].sum()

# 可视化
plt.figure(figsize=(10, 6))
plt.bar(monthly_sales.index, monthly_sales.values, color='skyblue')
plt.title('Total Sales by Month')
plt.xlabel('Month')
plt.ylabel('Sales ($)')
plt.xticks(monthly_sales.index)
plt.grid(axis='y')
plt.show()
```

### Q2: 哪个城市的订单量最大？

```python
city_orders = df['City'].value_counts()

plt.figure(figsize=(8, 8))
plt.pie(city_orders, labels=city_orders.index, autopct='%1.1f%%', startangle=140)
plt.title('Order Distribution by City')
plt.show()
```

### Q3: 什么时间段是用户下单的高峰期？

我们提取小时数据。

```python
df['Hour'] = df['Date'].dt.hour
hourly_orders = df.groupby('Hour')['Order ID'].count()

plt.figure(figsize=(12, 6))
plt.plot(hourly_orders.index, hourly_orders.values, marker='o')
plt.title('Orders by Hour of Day')
plt.xlabel('Hour (0-23)')
plt.ylabel('Number of Orders')
plt.grid(True)
plt.xticks(range(0, 24))
plt.show()
```
*分析结论*：通过观察折线图，如果晚上 19:00-21:00 是峰值，我们可以建议在这个时间段投放广告。

### Q4: 哪种产品最畅销（按销量 vs 按销售额）？

```python
# 按销量
product_quantity = df.groupby('Product')['Quantity'].sum().sort_values(ascending=False)

# 按销售额
product_revenue = df.groupby('Product')['Total Sales'].sum().sort_values(ascending=False)

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(15, 6))

ax1.bar(product_quantity.index, product_quantity.values, color='green')
ax1.set_title('Top Selling Products (Quantity)')
ax1.tick_params(axis='x', rotation=45)

ax2.bar(product_revenue.index, product_revenue.values, color='orange')
ax2.set_title('Top Revenue Products (Sales)')
ax2.tick_params(axis='x', rotation=45)

plt.show()
```
*分析结论*：可能 `Mouse` 销量最高（便宜），但 `Laptop` 贡献了最多的销售额。

## 4. 总结

通过这个简单的流程，我们展示了数据分析的标准范式：
1. **获取数据** (Load)
2. **清洗数据** (Clean)
