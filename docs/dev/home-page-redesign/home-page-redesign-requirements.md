## 4. 技术约束

### 4.1 技术栈

- **前端框架**：Vue 3 + TypeScript
- **UI 组件库**：Element Plus（可选使用）
- **动画库**：
  - 粒子效果：particles.js 或 tsparticles
  - 动画库：GSAP 或 Anime.js（用于复杂动画）
  - CSS 动画：用于简单的过渡和悬停效果
- **路由**：Vue Router（已有）
- **主题管理**：使用现有的主题系统或新增主题切换功能

### 4.2 性能要求

- **首屏加载时间**：< 2 秒（包含动画资源）
- **动画帧率**：保持 60fps
- **移动端优化**：
  - 简化粒子数量（桌面端 100+ 粒子，移动端 30-50 粒子）
  - 禁用鼠标跟随效果
  - 使用 CSS 动画替代 JS 动画（性能更好）

### 4.3 浏览器兼容性

- **现代浏览器**：Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **移动浏览器**：iOS Safari 14+, Chrome Mobile 90+
- **不支持**：IE 11 及以下

### 4.4 响应式断点

- **桌面端**：>= 1024px（完整动画和特效）
- **平板端**：768px - 1023px（中等复杂度动画）
- **手机端**：< 768px（简化动画，优化布局）
